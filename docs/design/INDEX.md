agent state, real-job evidence, and local publication ledgers are intentionally
kept out of the open-source tree.

## Notes

- Local task state, real run evidence, creator ledgers, and publication session
  records belong in ignored local files, not in public documentation.
- Historical design drafts should be consolidated into the public docs above
  before being added back to the repository.
